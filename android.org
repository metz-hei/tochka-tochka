Kotlin для новых проектов. Возможна поддержка проектов на Java.

Основной стек:

Koin - инъекции зависимостей.
Moxy - основа MVP.
Cicerone - навигация между экранами.
Room, ObjectBox - локальные базы данных.
Kotlin coroutines - многопоточность (возможна поддержка проектов с ReactiveX).
Retrofit - работа с сетью.

Архитектура:

Clean Architecture - базовое разделение на модули.
MVP - архитектура UI слоя (возможна поддержка проектов с MVVM).

Собственные библиотеки:

common-utils-android - сборник утилит различного назначения.
pagination-utils-android - обработка данных с постраничной загрузкой.
navigation-utils-android - навигация между экранами (обертка для Cicerone).
permissions-utils-android - управление разрешениями (доступ к памяти, камере и т.п.) (обертка для Dexter).
dialog-utils-android - простой вывод диалогов с настраиваемым отображением.
rubber-picker-ui-android - полоса выбора с 2 ползунками и анимацией.
circle-progress-bar-ui-android - круглая полоса прогресса с анимацией.

Glide (Picasso) - загрузка изображений.
FastAdapter - отображение списков.
Lottie - показ сложных анимаций.
com.github.navinilavarasan:rxWebSocket - работа с сокетами.
