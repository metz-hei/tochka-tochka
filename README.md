Это небольшая методичка для начинающих менеджеров по продукту или продуктовых дизайнеров. Для тех, кто пришел в новую команду или собирает команду для нового проекта.
Методичка поможет познакомиться с проектом, с внутренними регламентами, с устройством команды, поможет понять компетенции участников, наладить работу команды и принести пользу компании.

Некторые вопросы предлагают варианты решения, другие — нет. Анализируйте ответы, упрощайте жизнь себе и своей команде, думайте о пользе для бизнеса.
